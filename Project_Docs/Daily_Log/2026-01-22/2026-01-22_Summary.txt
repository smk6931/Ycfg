2. 최종 선정 및 사용 API (서비스 적용)
--------------------------------------------------

1) Nate (한국)
   - 용도: 실시간 검색어 키워드 (Top 20)
   - 이유: 현재 한국에서 가장 정확한 실시간 이슈 단어 제공

2) Yahoo! Japan (일본)
   - 용도: 실시간 검색어 키워드 (Top 20)
   - 이유: 일본 내 포털 점유율 높음, 정확한 실시간 순위 제공

3) Reddit (미국/글로벌)
   - 용도: 실시간 핫 토픽 (문장형 키워드)
   - 이유: 구글 트렌드 대체제, 전 세계 바이럴 이슈 파악 용이

4) Google News RSS (공통)
   - 용도: 뉴스 기사 콘텐츠 리스트
   - 이유: 키워드 검색 결과에 대한 관련 뉴스 확보용

5) YouTube Data API (공통)
   - 용도: 동영상 콘텐츠 리스트
   - 이유: 키워드 관련 영상 확보용
