
3. 프로젝트 발전 방향 (제안)

지금 프로젝트를 이렇게 확장해 나가시면 됩니다:

단계 1: 데이터 강화 (현재 진행 중)

키워드별 Keyword Suggestion이나 Related Queries를 추가 수집하여 풍부한 문맥 확보.
AI 요약 기능 추가: "왜 떴는지" 설명해주는 기능. (이게 있어야 콘텐츠 생성이 가능합니다)
단계 2: 스코어링 고도화 (선별)

지금은 단순 개수(news_count)지만, 여기에 Google Trends Score(0~100)와 최신성(Published At) 가중치를 더해 자체 랭킹 알고리즘을 만듭니다.
단계 3: 콘텐츠 생성 (발행 준비)

Threads용: AI 요약본을 바탕으로 "친근한 말투"로 변환.
인스타용: 요약된 핵심 문장 3개를 뽑아, Python(Pillow 라이브러리)이나 HTML-to-Image 기술로 배경 이미지 위에 글자를 합성해서 카드뉴스 이미지를 자동 생성합니다.
