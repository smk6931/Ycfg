1. 개요
================================================================================
기존 n8n(자동화 툴) 기반의 단순 수집 방식을 넘어, **"자체 AI 데이터베이스 구축 및 SaaS 서비스 확장"**을 목표로 개발 방향을 고도화합니다.


2. 핵심 전략: Hybrid System
================================================================================

A. 데이터 수집 효율화 (비용 최적화)
   - 전략: [무료 API]와 [유료 API]를 동시 가동하여 성능 비교.
   - 무료: Google RSS, Naver DataLab, YouTube API (비용 0원).
   - 유료: Apify 정밀 데이터 (성과 검증 후 유지 여부 결정).
   - 기대효과: 데이터 품질은 높이되, 불필요한 고정 비용 절감.

B. AI 자산화 (Vector DB)
   - 전략: 수집된 트렌드 키워드를 단순 텍스트가 아닌 '벡터(AI 데이터)'로 변환하여 저장.
   - 기대효과: "IT 관련 트렌드만 찾아줘" 같은 맥락 기반의 AI 검색 및 분석 가능.


3. 중장기 확장 로드맵 (Service Vision)
================================================================================

Phase 1. 데이터 파이프라인 구축 (현재)
   - Python 기반 수집 엔진 개발.
   - PostgreSQL 데이터베이스 구축.

Phase 2. 시각화 및 대시보드 (Front-end)
   - React 기반의 관리자 웹 페이지 개발.
   - 키워드 랭킹 및 급상승 추이 그래프 시각화.

Phase 3. 플랫폼 사업화 (SaaS)
   - 타 플랫폼(인스타/스레드/블로그) 계정 연동 기능 개발.
   - 클릭 한 번으로 3개 플랫폼에 자동 변환/업로드하는 '마케팅 허브'로 발전.
