================================================================================
Light Project Plan: 비용 0원 트렌드 분석기
================================================================================

작성일: 2026-01-20
목적: 복잡함 제거, 비용 0원, 핵심 기능 우선 구현


1. 수집 전략 (핵심 소스 4대장)
================================================================================

전제 조건: 모든 API와 도구는 "무료"만 사용한다.

A. Google Trends RSS (전세계 1순위)
--------------------------------------------------
- 특징: 전 세계 트렌드 파악의 핵심. 가장 강력하고 무료.
- 활용: 한국(KR), 미국(US), 일본(JP) 등 원하는 국가 "급상승 검색어" 추출.
- 방법: RSS URL만 있으면 끝. API 키도 필요 없음.
- 비용: 0원

B. Naver DataLab API (한국 1순위)
--------------------------------------------------
- 특징: 한국인 관심사 파악에 필수. 쇼핑/검색어 트렌드 제공.
- 활용: 
  1) 쇼핑인사이트: "지금 사람들이 뭘 사고 싶어하는지" (아이템 소싱용)
  2) 검색어트렌드: 특정 키워드의 검색 추이 그래프 확인.
- 방법: 네이버 개발자 센터 무료 가입 (하루 25,000회 무료).
- 비용: 0원

C. YouTube Data API v3 (영상 트렌드)
--------------------------------------------------
- 특징: 인스타/틱톡 트렌드의 원천. 영상 조회수 떡상 파악.
- 활용: 'Most Popular' API로 국가별 인기 영상 Top 50 추출.
- 방법: 구글 클라우드 콘솔에서 API Key 발급 (하루 10,000점 무료).
- 비용: 0원

D. 실시간 이슈 크롤링 (보조 소스)
--------------------------------------------------
- 대상: 네이트(Nate) 판/랭킹뉴스, 줌(Zum) 뉴스
- 이유: Signal.bz 같은 사설 사이트는 불안정함. 포털의 뉴스 랭킹이나 커뮤니티 화제글이 더 확실함.
- 활용: "지금 당장" 사람들이 뉴스 기사로 많이 보는 키워드 추출.
- 방법: Python requests + BeautifulSoup (가벼운 크롤링).
- 비용: 0원

*참고: 인스타/페이스북/스레드는 무료 수집이 매우 어렵고(계정 정지 위험), 유튜브와 구글 트렌드만으로도 충분히 커버 가능하므로 1단계에서는 제외.*


2. 단계별 개발 로드맵
================================================================================

1단계: 엑셀 파일 생성기 (이번 목표)
--------------------------------------------------
- 기능: 버튼 누르면 → Google/Naver/Youtube 싹 긁어서 → `trend_report.xlsx` 파일 생성.
- 구조: DB 없음, 서버 없음. 그냥 파이썬 스크립트 파일(.py) 하나로 실행.
- 출력물:
  [엑셀 시트 1] 구글_급상승_트렌드 (키워드, 관련뉴스, 국가)
  [엑셀 시트 2] 유튜브_인기영상 (제목, 조회수, 채널명, 링크)
  [엑셀 시트 3] 네이트_실시간_이슈 (뉴스제목, 링크)

2단계: 키워드 통합 및 알림
--------------------------------------------------
- 기능: 1단계 데이터에서 "공통적으로 등장하는 키워드" 찾기.
- 예: 구글에도 뜨고 + 유튜브에도 뜬 키워드 = "진짜 핫한 키워드" ★표시.
- 출력: 텔레그램이나 슬랙으로 "오늘의 핫 키워드: OOO" 알림 보내기.

3단계: 자동화 및 확장 (나중 목표)
--------------------------------------------------
- 그때 가서 서버(AWS 등) 도입 고민.
- 인스타 크롤링 등 고난이도 작업 추가.


3. 필요한 준비물 (사용자 준비)
================================================================================

1. Python 설치 (이미 되어 있음)
2. Google 계정 (유튜브 API용)
3. Naver 아이디 (개발자 센터 API 신청용 - 바로 승인됨)


4. 폴더 구조 (심플)
================================================================================
